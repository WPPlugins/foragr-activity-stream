Foragr gathers the activity from Facebook, Twitter, Disqus, IntenseDebate, and WordPess and unifies them in one or more activity streams on your website.

== Description ==

Foragr is a service and tool to aggregate activity streams.
Foragr gathers all the activity from components on your site and unifies them in one activity stream.
You can then build powerful community around it on your website.

The Foragr for WordPress plugin seamlessly integrates using the Foragr API and by syncing with WordPress plugins.

= Foragr for WordPress =

* Uses the Foragr API
* Tracks activity from Facebook, Twitter, Disqus, IntenseDebate, and WordPress so it can be viewed by you and your users
* Incentives built on top of this activity stream

= Foragr Features =

* Simple installation
* All site activity tracked in one place
* Completely customizable widgets
* Integrated web dashboard
* Powerful REST APIs
* RSS and Atom feeds for your activity stream
* Build community around your content

== Installation ==

1. Unpack archive to the 'wp-content/plugins/' directory inside
   of WordPress

  * Maintain the directory structure of the archive (all extracted files
    should exist in 'wp-content/plugins/foragr-activity-stream/')

2. From your blog administration, click on Plugins -> Foragr to change settings
   (WordPress 2.0 users can find the settings under Options > Foragr.)
